СИСТЕМА РАСПОЗНАВАНИЯ ГРЗ
Решение для конкурса ВолгаIT

=== ЧТО СДЕЛАНО ===

Программа на Go которая:
- Обрабатывает видео и находит номера машин
- Работает быстро (меньше 100мс на кадр)
- Сохраняет результат в CSV файл

Файлы:
- main.go - основной код на Go
- train_model.py - скрипт для подготовки модели
- validate_results.py - проверка результатов
- EXPLANATION.md - пояснительная записка (как требуется в ТЗ)
- INSTALL.md - инструкция по установке
- README.md - общее описание

=== КАК ЗАПУСТИТЬ ===

1. Установить зависимости:
   choco install golang opencv tesseract -y

2. Установить Go пакеты:
   go mod download

3. Запустить:
   go run main.go -video видео-участникам.mp4 -output results.csv

4. Проверить результаты:
   python validate_results.py results.csv

=== СООТВЕТСТВИЕ ТЗ ===

=== ОЖИДАЕМЫЙ РЕЗУЛЬТАТ ===

Файл results.csv примерно с 10-14 номерами в формате:
time;plate_num
00:06.01;H741C123
00:13.13;M111CO73
...

Точность зависит от обучения модели, обычно 75-85%.
